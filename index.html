Full-Stack Developer

A Standardized Grading System for Text Content

My NZSE capstone project, "Developing a Standardized Grading System for Text Content using Large Language Models," was amongst my most enriching project experiences. The project aim was to design an AI-based web application that grades the appropriateness of text content for people under the age of 15 on the grounds of explicit material, mature themes, emotional impact, and cultural sensitivity. I constructed the entire web application using Flask and Python, integrating advanced natural language processing (NLP) techniques for powering automated content analysis and categorization. The platform provides a scalable means to enable content creators and platform moderators to determine and sift through user-generated content before publication, helping improve safer online spaces for younger audiences. This project continued to build my full-stack development, NLP implementation and ethical AI design capabilities—capabilities I'm eager to apply in future professions as a developer or data analyst.
